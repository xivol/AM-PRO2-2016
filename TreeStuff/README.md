# Лабораторная работа №2
### Динамические структуры данных. Двоичные деревья

1. Реализовать тест для функции определения количества листьев в дереве.
2. Определить глубину дерева.
3. Определить количество узлов на заданной глубине.
4. Определить ширину дерева.
5. Проверить два дерева одинаковой структуры на равенство.
6. Создать копию дерева.
7. Удалить все листья с заданным значением из дерева.

# Лабораторная работа №3
### Динамические структуры данных. Деревья поиска
Используя структуру двоичного дерева, выполнить:
1. Построить дерево поиска.
2. Найти элемент с заданным значением в дереве (возвращает указатель на найденный элемент или nullptr, если элемент с заданным значением не найден).
3. Поиск с включением (возвращает указатель на найденный элемент или добавляет элемент в соответствующее место и возвращает указатель на него).
4. Посчитать количество вхождений заданного элемента в дерево.
5. Удалить первый элемент с заданным значением из дерева.
