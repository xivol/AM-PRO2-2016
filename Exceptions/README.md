# Лабораторная работа №10
### Классы. Исключения. Обработка исключений

Реализовать класс __динамический массив вещественных чисел__.

Класс должен содержать методы:
* конструкторы: с параметром, по умолчанию, копирования;
* деструктор;
* получения, изменения количества элемемнтов;
* добавление элемента в конец массива;
* вставка элемента в заданное место;
* удаление заданного элемента;
* удаление элемента с заданным индексом;
* поиск заданного элемента.

В классе должны быть перегружены операции:
* присваивания;
* доступа к элементам (`operator[]`);

Реализовать обработку ошибок с помощью исключений. 

Различные типы ошибок должны выбрасывать различные исключения.

Описать тесты методов класса.

# Лабораторная работа №11

Реализовать встроенный __класс исключений__ для массива. Создать исключения-наследники для различных ошибок.

Описать класс __вектор__, на базе динамического массива.
Закрыть модифицирующие методы класса-родителя.

Класс должен содержать:
* конструктор с указанием размера;
* конструктор с указанием размера и значения для инициализации всех элементов вектора;
* конструктор копии.
* метод вычисления нормы вектора.

Для класса вектор перегрузить операции:
* присваивания;
* сложения, вычитания векторов;
* умножения вектора на число;
* умножения векторов (скалярное произведения);
* сравнения векторов на равенство;
* сравнения векторов по норме.

Обратите внимание, что любые операции возможны только над векторами одной размерности. Предусмотреть возбуждение соответствующих исключений. 

Реализовать тесты для класса вектор
