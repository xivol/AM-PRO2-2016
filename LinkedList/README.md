# Лабораторная работа №1
### Динамические структуры данных. Двусвязные Списки

1. Посчитать количество элементов с заданным значением в списке.
2. Удалить заданное количество элементов, перед первым элементом с заданным значением.
3. Вставить заданное значение после каждого элемента с другим заданным значением.
4. Описать функцию сортировки списка (например, вставками).
5. Описать функцию поиска медианы списка.
6. Описать функцию двоичного поиска элемента в списке.
