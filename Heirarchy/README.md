# Лабораторная работа №9
### Классы. Наследование
По материалам [лабораторной работы №7](../CopyCtor):
* Наследовать класс _треугольник_ от многоугольника.
* Реализовать интерфейс треугольника из [лабораторной №4](../Class). 
* Изменить классы так, чтобы треугольник мог хранить координаты точек в массиве.

Реализовать класс _правильный многоугольник_ на базе многоугольника. 

Класс должен содержать:
* конструктор с параметрами (количество вершин, длина стороны, координаты центра).
* закрытые поля: длина стороны, центр;
* методы доступа к полям.
* переопределение методов вычисления:
 * центра;
 * площади;
 * периметра;
 * выпуклости;
* методы вычисления:
 * радиуса вписанной окружности;
 * радиуса описанной окружности.

Реализовать класс _прямоугольник_ на базе многоугольника. 

Класс должен содержать:
* конструктор с двумя параметрами(левая верхняя вершина, правая нижняя вершина);
* конструктор с тремя параметрами(левая верхняя вершина, ширина, высота);
* закрытые поля: ширина и высота;
* методы доступа к закрытым полям.
* переопределение методов вычисления:
 * центра;
 * площади;
 * периметра;
 * выпуклости;
* методы вычисления:
 * радиуса вписанной окружности (выбрасывать исключение, если не квадрат);
 * радиуса описанной окружности.

**Используя объекты данных классов решить задачу:**

Дан текстовый файл в формате:
    m
    P n xc yc len
    T x1 y1 x2 y2 x3 y3
    R x1 y1 x2 y2 
    …
Где `m` – задаёт количество многоугольников в файле; `P, T, R` –тип многоугольника.

`P` – правильный многоугольник:
* `n` – количество сторон;
* `xc yc` –координаты центра;
* `len` – длина стороны.

`T` – треугольник:
* `x1 y1 x2 y2 x3 y3` – координаты вершин.

`R` – прямоугольник:
* `x1 y1 x2 y2` – координаты левой верхней и правой нижней вершины.

Считать данные из файла и вывести площадь и периметр каждой из фигур.
