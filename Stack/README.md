#Лабораторная работа №14
###Абстрактные типы данных. Стек

Реализовать класс __стек целых чисел__ на базе динамического массива и списка.

Класс должен содержать методы:
* конструктор по умолчанию, копирования;
* деструктор;
* проверка стека на пустоту;
* добавление в стек;
* извлечение элемента из стека;
* просмотр вершины стека.

Для класса перегрузить операции:
* присваивания,
* вывод в поток.

Сравнить время выполнения 1 000 000 операций поочередной вставки и удаления элементов для обеих реализаций стека.
