# Лабораторная работа №4
### Классы. Поля и методы
Реализовать класс точки на плоскости, с закрытыми полями x,y. Класс должен содержать:
* конструктор с параметрами (x,y)
* конструктор по умолчанию
* методы доступа к полям
* метод вычисления расстояния до другой точки
* метод вычисления [расстояния от точки до прямой](http://algolist.manual.ru/maths/geom/distance/pointline.php)

Используя объекты этого класса описать класс треугольник. Класс должен содержать:
* закрытые поля для вершин;
* конструктор с параметрами (p1, p2, p3);
* методы доступа к полям;
* методы вычисления:
 * типа треугольника (по углу);
 * периметра;
 * площади;
 * [радиуса вписанной окружности;](https://ru.wikipedia.org/wiki/%D0%92%D0%BF%D0%B8%D1%81%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BE%D0%BA%D1%80%D1%83%D0%B6%D0%BD%D0%BE%D1%81%D1%82%D1%8C)
 * [радиуса описанной окружности.](https://ru.wikipedia.org/wiki/%D0%9E%D0%BF%D0%B8%D1%81%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BE%D0%BA%D1%80%D1%83%D0%B6%D0%BD%D0%BE%D1%81%D1%82%D1%8C)

Описать функции ввода и вывода треугольника. Описать тестирующий класс для методов треугольника. При описании конструктора необходимо проверять, что треугольник не вырожденный.
